# Create an exercise

Exercises are defined in `/packages/exercises`.
An exercise definition requires three things:

1. The schema
2. Feedback functions, to grade and provide feedback
3. The views

## Schema

An **exercise schema** describes how an exercise is structured
and how it is stored in the database.

- **name**: a unique identifier name for the exercise.
- **question**: an object which specifies the shape of the question.
- **steps**: an object which specifies the potential shape of the user input for each step of the exercise.

These informations are used by `@learning/core` for multiple purposes:

- Provide **type safety** for the definition of the feedback and the views
- **Validate** the data and provide useful errors.
- Give information on how to **display input fields** in the views.

```tsx
export const schema = {
  name: 'math/factor',
  question: {
    expr: field('math').meta({
      label: 'Expression',
      description: "L'expression à factoriser",
    }),
  },
  steps: {
    start: {
      attempt: field('math').meta({
        label: 'Réponse',
        description: 'Entrez votre réponse ici',
      }),
    },
    binomial: {
      type: field('select').options({
        squaredSum: '(a + b)^2',
        squaredDiff: '(a - b)^2',
        conjugate: '(a + b)(a - b)',
      }),
    },
    root: { root: field('math') },
  },
} as const satisfies Schema
```
